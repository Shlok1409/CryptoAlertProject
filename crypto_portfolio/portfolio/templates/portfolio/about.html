About the project

Generic Crypto-Portfolio started as a project for Python in the enterprise course. Website is written entirely using Django framework. It's supposed to help users manage their cryptocurrency portfolios by providing a place to store information about their, assets, display important statistics regarding their choice of currencies and plotting all the data. What makes the website special is price prediction algorithm which uses very sophisticated machine learning model and provides a hint at potential future changes in cryptocurrencies' prices.

Authors: Michał Orlewski, Damian Raczyński, Kamil Kaproń
